## 와이어프레임 파일 목록

### 1. 서비스 발견 화면

#### 1-1. 게이트웨이 (서비스홈)
- **파일**: `1-1-서비스 발견 화면_게이트웨이.html`
- **설명**: 게이트웨이(서비스홈)에서 지킴진단 서비스를 발견하는 화면
- **주요 요소**: 서비스 소개 배너, CTA 버튼, 주요 기능 안내

#### 1-2. 매물 상세 페이지
- **파일**: `1-2-서비스 발견 화면_매물 상세.html`
- **설명**: 매물 상세 페이지에서 지킴진단 서비스를 발견하는 화면
- **주요 요소**: 배너, CTA 버튼, 매물 정보 연동

### 2. 진단 시작 화면

#### 2-1. 게이트웨이에서 시작
- **파일**: `2-1-지킴진단 시작화면_게이트웨이에서 시작.html`
- **설명**: 게이트웨이에서 진단을 시작하는 경우, 매물 정보를 직접 입력하는 화면
- **주요 요소**: 주소/면적 직접 입력, 보증금, 월세, 계약 기간 입력, 가격 안내, 약관 동의

#### 2-2. 매물 상세에서 시작
- **파일**: `2-2-지킴진단 시작화면_매물 상세에서 시작.html`
- **설명**: 매물 상세 페이지에서 진단을 시작하는 경우, 매물 정보가 자동으로 입력된 화면
- **주요 요소**: 주소/면적 자동 완성, 보증금, 월세, 계약 기간 입력, 가격 안내, 약관 동의

### 3. 결제 화면 (참고용)
- **파일**: `3-결제하기.html`
- **설명**: 토스 브랜드 페이를 통한 결제 진행 화면 (참고용)
- **주요 요소**: 매물 정보 요약, 결제 금액, 결제 수단 선택, 결제 버튼
- **⚠️ 중요**: 결제 진행 화면은 토스 브랜드 페이 API 연동으로 처리되므로, 실제 구현 플로우에서는 별도 화면으로 구현하지 않고 토스 결제 팝업으로 처리됩니다.

### 4. 리포트 생성 대기 화면
- **파일**: `4-리포트 생성 대기 화면.html`
- **설명**: 리포트 생성 중 대기 상태를 표시하는 화면
- **주요 요소**: 진행 상황 표시, 예상 소요 시간, 단계별 진행 상태

### 5. 리포트 상세 화면
- **파일**: `5-리포트 상세 화면.html`
- **설명**: 종합 진단 리포트를 확인하는 화면
- **주요 요소**: 종합 안전 등급, 진단 항목별 상세 정보, 특약 조항, PDF 다운로드

### 6. 나의 지킴 진단 리포트
- **파일**: `6-나의 지킴 진단 리포트.html`
- **설명**: My 직방에서 발급받은 리포트 목록을 확인하는 화면
- **주요 요소**: 리포트 목록, 리포트 항목, 진단 상태 표시

### 7. 마이페이지 - 지킴진단 내역
- **파일**: `7-마이페이지-지킴진단내역.html`
- **설명**: 마이페이지 내 지킴진단 서비스 섹션
- **주요 요소**: 사용자 프로필, 진단 내역 통계, 빠른 접근 메뉴
